Bootstrap

使用世界上最流行的前端组件库在Web上构建响应式，移动优先的项目。

bootstrap特点

Bootstrap是一个用于使用HTML，CSS和JS开发的开源工具包。使用我们的Sass变量和mixins，响应式网格系统，大量的预构建组件以及基于jQuery构建的强大插件，可以快速创建您的想法的原型，或构建您的整个应用程序。

npm 安装

通过npm，Composer或Meteor包含Bootstrap的源Sass和JavaScript文件。程序包管理的安装不包括文档，但包括我们的构建系统和自述文件。

npm install bootstrap gem install bootstrap -v 4.3.1

引导cdn

当您只需要包含Bootstrap的已编译CSS或JS时，可以使用BootstrapCDN。

Bootstrap CDN

npm 使用介绍

npm 同node一同安装，打开cmd命令窗口。

npm 命令安装模块

$ npm install <Module Name>

查看安装模块的信息

$npm list -g 全局安装的模块信息 $npm list <Module Name> 指定模块的版本号

卸载模块

$ npm uninstall <Module Name>

查看包在node_modules目录下是否存在

$ npm ls

更新模块

$ npm update express

搜索模块

$ npm search express

创建模块

$ npm init

在 npm 资源库中注册用户

$ npm adduser

发布模块

$ npm publish

Element

Element，一套为开发者、设计师和产品经理准备的基于 Vue 2.0 的桌面端组件库

Element特点

与现实生活的流程，逻辑保持一致，并遵循用户习惯的语言和概念；所有的元素和结构需保持一致;通过界面样式和交互动效让用户可以清晰的感知自己的操作;操作后，通过页面元素的变化清晰地展现当前状态。设计简洁直观的操作流程；语言表达清晰且表意明确，让用户快速理解细长做出决定；界面简单直白，让用户快速识别而非回忆，减少用户记忆负担。用户可以自由的进行操作，包括取消，回退和终止当前操作等。

npm 安装

npm i element-ui -S
